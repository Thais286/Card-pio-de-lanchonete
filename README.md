# Card-pio-de-lanchonete
Projeto focado em html e css, nele coloco em prática meus aprendizados adquiridos no curso em vídeo do professor Gustavo Guanabara . O projeto é sobres  o cardápio de uma lanchonete onde pode-se encontrar valores dos lanches e tamanhos,entre eles estão pizzas,hambúrgues,sobremesas e bebidas.
